GoBus Application
------------------
The app displays the current location of the user and it also displays the location of all the buses near the current location. These details will be displayed when the application is launched.The bus markers have their details on them and it can be viewed when the marker icon is clicked. The user will also be able to  come back to the same state where the app was left while closing the application even if the application was killed or closed.
---------------------------------------------------------------------------------------------------------
Functionality
-------------
1.On launch, a map is displayed, overlaid with indicators showing the current positions of buses. 
2.The marker for each bus includes the route number. 
3.The positions of the buses are updated approximately every 15 seconds. 
4.The user can choose to centre the map to their current location using a button.
5.The user can zoom and pan the map freely. 
6.When returning to the app, the map shows the same region it was showing when the app was paused or closed,even if the app is terminated and relaunched.
---------------------------------------------------------------------------------------------------------
Challenges
----------
1.The app cannot be accessed in offline mode.
